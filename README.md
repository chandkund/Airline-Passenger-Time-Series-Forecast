# Time-Series-Forecasting

This repository contains a structured project scaffold for time series forecasting with:

- `data/` for raw and processed data
- `notebooks/` for exploratory analysis and experiments
- `src/` for Python modules
- `outputs/` for plots and evaluation results

## Project Structure

Refer to the repository layout for source, data, and output organization.

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the command-line entrypoint:

```bash
python main.py
```

Run the Streamlit app:

```bash
streamlit run streamlit_app.py
```
